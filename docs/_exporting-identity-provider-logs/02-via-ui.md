If you have not done so already, please [log in to Federation Manager](/log-into-federation-manager/01-overview) to get started.
{: .callout-info }

- Then visit the identity provider you would like to export the logs of choose the start and end date and click export.

![Log Export UI](/assets/images/exporting-identity-provider-logs/identity-provider-log-export-ui.png)

- Please note, depending on the date range this can take some time, once its completed you will get a notification via email and in the federation manager portal with a link to your logs.
