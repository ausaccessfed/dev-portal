The attribute `eduPersonAffiliation` (with permissible values including faculty, student, staff, alum, member, affiliate, employee, and library-walk-in) can be defined based on groups within a directory.

### What you'll learn

- How to use a script to define `eduPersonAffiliation` based on groups in a directory

### What you'll need

- Completion of the [SAML Integration Tutorial](/saml-integration/01-overview).
